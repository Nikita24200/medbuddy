MedBuddy AI Healthcare Assistant
📋 Description
MedBuddy is an AI-powered healthcare assistant that helps users understand and manage their prescriptions. The application allows users to upload prescription images or text, extracts medication information using advanced AI models, and provides a user-friendly interface to view and track their healthcare data.

✨ Features
User Authentication: Secure signup and login with JWT tokens
Prescription Upload: Support for image and text-based prescription input
AI-Powered Extraction: Intelligent medication information extraction using multiple AI models (OpenAI, Google GenAI, Hugging Face, Together AI)
Rule-Based Fallback: Reliable extraction using pattern matching when AI is unavailable
Prescription Management: Store and view prescription history with diagnosis, medications, side effects, and follow-up instructions
Multi-language Support: Interface supports multiple languages
Responsive Design: Modern, mobile-friendly UI with dark mode support
Real-time Processing: Fast prescription analysis and storage
🛠 Tech Stack
Backend
Node.js - Runtime environment
Express.js - Web framework
Supabase - Database (PostgreSQL) and authentication
JWT - Authentication tokens
Multer - File upload handling
CORS - Cross-origin resource sharing
AI/ML
OpenAI - Primary AI model for text analysis
Google GenAI - Alternative AI processing
Hugging Face - Machine learning models
Together AI - Distributed AI computing
Frontend
HTML5/CSS3 - Core markup and styling
JavaScript (ES6+) - Client-side logic
React - UI framework (via CDN)
Babel - JavaScript transpilation
🚀 Installation
Prerequisites
Node.js (v14 or higher)
npm or yarn
Supabase account and project
